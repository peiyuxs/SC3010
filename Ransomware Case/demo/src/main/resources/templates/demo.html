🔐 How Real Ransomware Works:

1. Infection: Malware executes on the victim's computer
2. Key Generation: Generates unique AES-256 encryption key for each file
3. Encryption: Encrypts files in-place (original filename preserved)
4. Key Exfiltration: Sends encryption keys to attacker's command & control server
5. Ransom Note: Displays ransom demand with payment instructions
6. No Recovery: Without the key, AES-256 encrypted files are unrecoverable

⚠️ This Demo Shows:
• Files are encrypted with real AES-256 (military-grade encryption)
• Original filenames are preserved (realistic behavior)
• Encrypted files contain binary data that cannot be read
• Keys are displayed in web interface for educational purposes
• In real ransomware, keys would NOT be accessible to the victim

🛡️ Protection Tips:
• Maintain offline backups of important files
• Keep software and antivirus updated
• Be cautious with email attachments and links
• Use the principle of least privilege for user accounts
• Never pay ransom demands - it encourages more attacks
