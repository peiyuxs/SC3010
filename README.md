## SQL Injection Demo quick start

Full documentation is in [sql-injection/README.md](sql-injection/README.md).

From [sql-injection](sql-injection):

1. `npm install`
2. `cp .env.example .env.local`
3. `docker compose up -d`
4. `npm run db:seed`
5. `npm run dev`

Open `http://localhost:3000`.
